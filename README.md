A role-based Complaint Management System built with React.

## Run Project

### 1. Install Dependencies

```bash
npm install
```

### 2. Start Development Server

```bash
npm run dev
```

### 3. Open Browser

Open the URL shown in the terminal, usually:

```text
http://localhost:5173
```

---

## Technologies Used

- React
- JavaScript (ES6)
- CSS3
- Vite
- Local Storage

---

## Features

### Student
- Login
- Submit Complaint
- View Complaint History
- Track Complaint Status
- Update Profile

### Admin
- Login
- View All Complaints
- Update Complaint Status
- Manage Complaints
